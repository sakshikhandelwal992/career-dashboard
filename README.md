# 🚀 Career Compass – Student Career Dashboard

A modern and responsive Career Dashboard that helps students organize their learning journey, track goals, monitor progress, and manage certificates in one place. This project demonstrates modular frontend architecture, responsive design, client-side routing, and deployment practices.

---

## ✨ Features

- 📌 Professional Dashboard Layout
- 🎯 Track Daily Learning Goals
- 📈 Learning Progress Section
- 🏆 Certificates Section
- 💡 Motivation Card
- 🌙 Light/Dark Mode Toggle
- 📱 Fully Responsive Design
- 🧩 Modular JavaScript Architecture
- 🔄 Client-Side Navigation
- 🚀 Live Deployment

## 🧠 Project Highlights

### ✔ Modular Frontend Architecture
The project follows a structured folder organization by separating HTML, CSS, JavaScript, and assets into dedicated directories. JavaScript functionality is divided into multiple modules, making the application easier to maintain and scale.

### ✔ Responsive User Interface
The dashboard is designed using CSS Grid and Flexbox with mobile-first responsive media queries, ensuring a smooth experience across desktop, tablet, and mobile devices.

### ✔ Client-Side Routing
Navigation is handled on the client side to provide seamless transitions between dashboard sections without requiring a full page reload.

### ✔ Dynamic Theme
Implemented Light and Dark Mode using reusable CSS Variables and JavaScript for an enhanced user experience.

### ✔ Performance Optimization
Optimized project assets through organized file management, reusable styles, and lightweight code structure to improve maintainability and loading performance.
